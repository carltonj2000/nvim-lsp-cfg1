## NeoVIM Configuration

### Code Creation

```bash
touch README.md
README.md
mkdir lua
touch lua/base.lua
touch lua/keymaps.lua
touch lua/plugins.lua
```

Then installed and configures the plugins in the following order.

- which-key
- telescope - file finder, greper, browser
- tokoyonight - color scheme
